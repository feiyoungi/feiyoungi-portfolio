Le Cat Café

UX/UI Design / App Design

Date:September 2023 – November 2023

Role: UX/UI Designer, Animator, UX Researcher for Heuristic Data

Tools: Figma

This project is a rebranding of the Le Cat Café's website accross a small, medium, and large screen for screen responsiveness. The primary goal of this rebranding is to declutter the text, improve the navigation system to be more intuitive, and push forth a cozy vibe with the use of softer colors and edges.
